# Extra

Off-roadmap learnings — things picked up along the way that don't belong to a
numbered week: side experiments, rabbit holes worth keeping, notes from talks or
papers outside the curriculum, small utilities. [← Modules](../README.md) · [Hub](../../README.md)

## Ground rules

- One directory per thing: `extra/<topic>/` with its own short README (what it is,
  why it was worth the detour, how to run it if it runs).
- Keep it honest — scratch quality is fine here; this is the margin of the notebook,
  not the coursework.
- If something in here grows into a real build, promote it to a numbered module
  ([recipe](../README.md#how-a-module-becomes-a-project)) and leave a pointer behind.
